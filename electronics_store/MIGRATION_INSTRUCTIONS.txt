ИНСТРУКЦИИ ПО МИГРАЦИЯМ
========================

1. Создание миграций:
   python manage.py makemigrations

2. Применение миграций:
   python manage.py migrate

3. Создание суперпользователя (для админки):
   python manage.py createsuperuser

ЧТО БУДЕТ СОЗДАНО В БАЗЕ ДАННЫХ:
================================

1. Таблица main_userprofile:
   - user_id (связь с auth_user)
   - patronymic (отчество)
   - phone (телефон)
   - birth_date (дата рождения)
   - address (адрес)
   - avatar (аватар)
   - created_at (дата создания)
   - updated_at (дата обновления)

2. Таблица main_usersession:
   - user_id (связь с auth_user)
   - session_key (ключ сессии)
   - ip_address (IP адрес)
   - user_agent (браузер)
   - created_at (дата входа)
   - last_activity (последняя активность)
   - is_active (активна ли сессия)

3. Автоматические сигналы:
   - При создании пользователя автоматически создается профиль
   - При сохранении пользователя обновляется профиль

ФУНКЦИОНАЛ:
===========

✅ Регистрация с полной валидацией
✅ Авторизация с отслеживанием сессий
✅ Профиль пользователя с дополнительными полями
✅ Админ-панель для управления пользователями
✅ Выход из системы с деактивацией сессий
✅ Отслеживание IP и браузера пользователей

АДМИНКА:
========

После создания суперпользователя доступна по адресу:
http://127.0.0.1:8000/admin/

В админке можно:
- Управлять пользователями и их профилями
- Просматривать активные сессии
- Редактировать дополнительную информацию
